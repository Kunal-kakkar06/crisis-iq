# CrisisIQ - AI Disaster Response Platform

CrisisIQ is a production-ready web dashboard built for high-stakes disaster response operations. It provides real-time situational awareness, resource management, and AI-driven fairness auditing.

## Tech Stack
- **Framework**: React 19 (Functional Components)
- **Styling**: Tailwind CSS 3.4
- **Animations**: Framer Motion
- **Charts**: Chart.js / React-Chartjs-2
- **Icons**: Lucide React
- **Theming**: Dark/Light mode support with context

## Project Structure
- `src/components`: Modular UI components (Sidebar, Header, Metrics, etc.)
- `src/context`: Theme and State management
- `src/data`: Mock JSON data for demonstration
- `src/styles`: Global CSS and Tailwind directives

## Features
- **Strict UI Fidelity**: Layout and spacing match the requested design inspiration.
- **Dual Themes**: Premium Navy Dark Dashboard and Clean Light SaaS Dashboard.
- **Fairness Engine**: Specialized module for monitoring allocation bias (Urban/Rural/Tribal).
- **Audit Logs**: Immutable log tracking for system and operator actions.
- **Resource Tracking**: Visual progress indicators for critical supplies.
- **Live Map**: Operations-focused map view with severity zones.

## Google Tools Integration (Placeholders)
- **Vertex AI**: Powering the "Severity Analytics" and bias scoring.
- **Gemini**: Used for report validation and audit log signing.
- **Google Maps**: Visualizing real-time disaster zones and resources.
- **Firebase**: Ensuring real-time updates across all panels.

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
